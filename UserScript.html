Обучающий режим «Новичок» предназначен для тех, кто только начинает работать с программой для заполнения документов. Он помогает освоить базовые навыки работы с интерфейсом программы и основными инструментами.
В режиме «Новичок» пользователь получает доступ к упрощённому интерфейсу, где представлены только самые необходимые инструменты. Кроме того, программа предоставляет подсказки и советы по использованию каждого инструмента, без ограничения по времени.
В процессе обучения пользователь выполняет ряд простых задач, таких как создание нового документа, выбор шаблона, ввод текста и редактирование его форматирования. По мере выполнения задач сложность постепенно увеличивается.
После завершения обучения пользователь может перейти в обычный режим работы с программой, где ему будут доступны все возможности и инструменты или пройти другой обучающий сценарий.
